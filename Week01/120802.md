## 📄 문제 

### [두 수의 합](https://school.programmers.co.kr/learn/courses/30/lessons/120802)

<br>

## 🧚🏻‍♀️ 코드 작성

```javascript
function solution(num1, num2) {
    return num1 + num2;
}
```

<br>

## 📝 User 풀이

```javascript
const solution = (num1, num2) => num1 + num2
```

<br>

## ✨ 소감

+ 화살표 함수를 이용하면 더 간단하게 구현할 수 있다는 것을 알게됐다.
